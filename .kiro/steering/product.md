# Product Overview

Alob Express Infrastructure is an Infrastructure-as-Code (IaC) solution for deploying a complete automation and AI stack on cloud platforms. The system provides a unified, modular installation framework for business automation tools.

## Core Services

- N8N: Workflow automation and orchestration platform
- Evolution API: WhatsApp integration service with S3 media storage
- Dify AI: LLM application development platform with vector storage
- Traefik: Reverse proxy with automatic SSL certificate management
- Portainer: Docker Swarm management interface

## Target Platforms

- AWS (Amazon Web Services)
- Google Cloud Platform (GCP)
- Single high-spec VM architecture (16GB RAM recommended)

## Key Features

- Automated Docker Swarm orchestration
- Multi-cloud support with automatic provider detection
- Modular installation system with checkpoint recovery
- Automatic DNS verification and SSL certificate generation
- S3-based backup and restore functionality
- Resource optimization (80/20 rule for memory allocation)
- Business-specific isolation (separate directories per deployment)

## Use Cases

- Business process automation
- WhatsApp chatbot integration
- AI-powered customer service
- Workflow orchestration
- Multi-tenant automation infrastructure
